## Getting started

```bash
# clone the project
git clone https://github.com/Dengjunle/vue-admin-demo-Le.git

# enter the project directory
cd vue-admin-demo-Le

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:4000

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
